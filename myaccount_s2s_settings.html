S2S Conversion Tracking

Example:http://mysite.com/s2s.php?adn=affilist&campaign=[ad]

Available tokens for S2S conversion tracking

Type Description
[conversion_id] Conversion / transaction ID (e.g. "15829637118")
[ad] ID of the ad (e.g. “123456789”)
[campaign] ID of the campaign (e.g. “12345”)
[price] Conversion value in campaign’s currency (e.g. “12.34”)
[usd_price] Conversion value in US Dollars (e.g. “12.34”)
[currency] Campaign’s currency (e.g. “EUR”)
[cv] Custom value - add "&cv=YOUR_DATA" to the tracking link/banner ad tag
[countries_code] Visitor’s country - country code (e.g. “AE”)
[language_code] Visitor’s language code (e.g. “AE”)
[placement_name] Placement name (set in the tracking link/ad tag)
[cities] Visitor’s city (e.g. “Cupertino”)
[regions] Visitor’s region/state (e.g. “California”)
[browsers] Visitor’s website browser (e.g. “Chrome 57”)
[carriers] Visitor’s mobile carrier (e.g. “Orange”)
[countries] Visitor’s country (e.g. “United Arab Emirates”)
[devices] Visitor’s device (e.g. “iPhone 6s Plus”)
[devices_types] Visitor’s device type (e.g. “Smartphone”)
[languages] Visitor’s language (e.g. “Arabic”)
[operating_systems] Visitor’s operating system (e.g. “Windows 10”)
[link_types] Use translator to describe link_types

Note:If the call of your S2S URL is unsuccessful because of an error on your website (e.g. error 404, 502, 504...) or because you website is not responding for more than 10 seconds, we will re-call your S2S URL after 5 minutes. In case of another failure we will try it again after 30 minutes and then again after 60 minutes (max. 4 calls in total).

After changing these settings, only new conversions generated after the change will be affected.

S2S Callback URL for clicks:

Example:http://mysite.com/s2s.php?adn=affilist&campaign=[cc]

Available tokens for S2S click tracking

Type Description
[cv] Custom value - add “&cv=YOUR_DATA” to the tracking link/banner ad tag
[cc] Custom value - add "&cc=YOUR_DATA" to the tracking link/banner ad tag
